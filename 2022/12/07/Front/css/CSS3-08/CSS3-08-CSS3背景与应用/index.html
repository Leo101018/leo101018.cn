CSS3背景与应用

CSS 可以添加背景颜色和背景图片，以及来进行图片设置。

background-color 背景颜色
background-image 背景图片地址
background-repeat 是否平铺
background-position 背景位置
background-attachment 背景固定还是滚动
背景的合写（复合属性）
background:背景颜色 背景图片地址 背景平铺 背景滚动 背景位置

背景图片(image)

语法：

参数：

none : 　无背景图（默认的）
url : 　使用绝对或相对地址指定背景图像

background-image 属性允许指定一个图片展示在背景中（只有CSS3才可以多背景）可以和 background-color 连用。 如果图片不重复地话，图片覆盖不到地地方都会被背景色填充。 如果有背景图片平铺，则会覆盖背景颜色。

小技巧： 我们提倡 背景图片后面的地址，url不要加引号。

背景平铺（repeat）

语法：

参数：

repeat : 　背景图像在纵向和横向上平铺（默认的）

no-repeat : 　背景图像不平铺

repeat-x : 　背景图像在横向上平铺

repeat-y : 　背景图像在纵向平铺

设置背景图片时，默认把图片在水平和垂直方向平铺以铺满整个元素。

repeat-x : 　背景图像在横向上平铺

repeat-y : 　背景图像在纵向平铺

设置背景图片时，默认把图片在水平和垂直方向平铺以铺满整个元素。

默认是平铺的，这里改成no-repeat，不平铺。

repeat-x是横向平铺。

repeat-y，纵向平铺。

背景位置(position)

语法：

参数：

length : 　百分数 | 由浮点数字和单位标识符组成的长度值。请参阅长度单位
position : 　top | center | bottom | left | center | right

说明：

设置或检索对象的背景图像位置。必须先指定background-image属性。默认值为：(0% 0%)。
如果只指定了一个值，该值将用于横坐标。纵坐标将默认为50%。第二个值将用于纵坐标。

注意：

  1. position 后面是x坐标和y坐标。 可以使用方位名词或者 精确单位。
  2. 如果和精确单位和方位名字混合使用，则必须是x坐标在前，y坐标后面。比如 background-position: 15px top; 则 15px 一定是 x坐标 top是 y坐标。

实际工作用的最多的，就是背景图片居中对齐了。

默认是left和top，就是左上角。

右上角。

就是属性值，bottom和right，这个谁在前，谁在后无所谓。

水平中间，垂直中间。

只写一个方位名词，另外一个默认就是居中的。

网页中的坐标。

我们也可以指定图片的精确坐标位置。

使用精确坐标。

如果使用纯方位名字的话，那么属性值的前后没有顺序的。

对于精确定位和混搭，属性值都是有顺序的，前面是x，后面是y。

背景简写

background属性的值的书写顺序官方并没有强制标准的。为了可读性，建议大家如下写：

background:背景颜色 背景图片地址 背景平铺 背景滚动 背景位置

背景位置，水平是center，垂直是100px。

注意，最后的center 100px这个背景位置是一体的，x，y坐标，他俩不能够分开的。然后其它的部分，位置顺序不是严格规定的。

背景缩放(CSS3)

通过background-size设置背景图片的尺寸，就像我们设置img的尺寸一样，在移动Web开发中做屏幕适配应用非常广泛。

其参数设置如下：

a) 可以设置长度单位(px)或百分比（设置百分比时，参照盒子的宽高）

b) 设置为cover时，会自动调整缩放比例，保证图片始终填充满背景区域，如有溢出部分则会被隐藏。我们平时用的cover 最多

c) 设置为contain会自动调整缩放比例，保证图片始终完整显示在背景区域。

我们知道，图片大小是固定的，那么图片作为盒子的背景图片的时候，能否跟着进行缩放呢？

在css3新特性中是可以支持的。

background-size，用来设置背景图片的尺寸。

img是插入图片，不是背景图片。

尽量不要宽和高一起改，因为不能很好的控制比例，因此，给定一个高或者宽即可，能够自动进行等比例缩放的。

还可以使用百分比的形式。

因此，可以使用精确单位，也可以使用百分比的方式。

前面的背景缩放，没有将div这个盒子充满。

可以手动的指定背景图片的宽高，充满盒子。

但是这样写，不方便，而且图片也失真了。

可以看到图片也没有失去比例。

缩放的方式是这样的，图片往下放大，高度和盒子一样高了，但是宽度还没有到，因此，继续等比例的放大。

直到宽度和盒子一样大小为止。

由于是等比例缩放的，所以，此时下面多出来的部分就被隐藏了。

图片正常比例显示，但是盒子有一部分漏出了。

contain是这样，等比例放大，然后首先发现高度满足了盒子的高度了，此时图片就不会再进行缩放了。因此，宽度还没有满足盒子的宽度，因此，背景色就会漏出。

如果图片是这样的，那么就是首先宽度满足了盒子了，那么此时高度就不会再变了，图片不会再进行缩放了。此时低下背景色就会漏出了。

我们平时使用cover较多。

多背景(CSS3)

以逗号分隔可以设置多背景，可用于自适应布局 做法就是 用逗号隔开就好了。

  • 一个元素可以设置多重背景图像。
  • 每组属性间使用逗号分隔。
  • 如果设置的多重背景图之间存在着交集（即存在着重叠关系），前面的背景图会覆盖在后面的背景图之上。
  • 为了避免背景色将图像盖住，背景色通常都定义在最后一组上，

我们是否可以在一个盒子上放置多张图片呢，如图，左上角放一张图片，右下角放一张图片。

此时两张背景图片是有交集的。有重叠的。

前面的将后面的覆盖了。
